Gaaner Opare

"Gaaner Opare" was a Visual Basic project in my 3rd semester of college. We were asked to create an application using VB which would maintain records of various music records available in an imaginary music store.

I used Visual Studio 2012 to create the interface and functionalities and also to link the database with the application. I used MS Access for the backend database. The application can be downloaded here.
